### 关闭页面

点击左上角的`x`，就关闭页面了：

* ![filza_closed_page](../../../assets/img/filza_closed_page.png)

### 新建页面

可以点击 底部中间的 `➕` 加号 去新建窗口。

默认打开的还是 

* `/var/mobile/Documents`

切换显示模式后：

* ![filza_new_page_mode](../../../assets/img/filza_new_page_mode.png)
